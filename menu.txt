메뉴 : 에스프레소(Espresso)
설명 : 정교한 온도와 압력으로 추출한 깊고 선명한 한 잔의 진미.
풍부한 크레마와 농밀한 바디감을 통해 원두 본연의 향미를 온전히 느낄 수 있는, 카페 소울만의 클래식한 시그니처 에스프레소입니다.
메뉴 : 카페 라떼(Caﬀé Latte)
설명 : 부드러운 스팀 밀크가 에스프레소와 조화를 이루며 우아한 질감과 은은한 단맛을 선사합니다.
고급 우유의 자연스러운 고소함이 깊은 에스프레소와 어우러져 누구에게나 사랑받는 부드러운 데일리 커피입니다.
메뉴 : 카푸치노(Cappuccino)
설명 : 고운 우유 거품 위에 담긴 진한 에스프레소의 풍미가 입 안 가득 퍼지는 정통 이탈리안 스타일.
첫 모금의 따뜻함과 마지막까지 이어지는 풍부한 향의 여운이 특별한 경험을 선사합니다.
메뉴 : 콜드 브루(Cold Brew)
설명 : 차가운 물로 시간의 정성을 더해 천천히 추출한 커피.
잡미 없이 깔끔하며, 은근하게 퍼지는 단맛과 부드러운 목넘김이 돋보이는 카페 소울의 베스트셀러입니다.
메뉴 : 바닐라 빈 라떼(Vanilla Bean Latte)
설명 : 마다가스카르산 바닐라 빈 시럽을 사용해 깊고 순수한 향을 담아낸 프리미엄 라떼.
실크처럼 부드러운 질감과 고급스러운 단맛이 조화를 이루며, 한 모금마다 따스하고 은은한 여운을 남깁니다.
메뉴 : 플랫 화이트(Flat White)
설명 : 에스프레소와 벨벳 같은 마이크로폼이 정교하게 어우러진 오스트레일리아식 프리미엄 커피.
라떼보다 진하고 카푸치노보다 부드러운 완벽한 밸런스로, 원두의 깊은 풍미를 우아하게 느낄 수 있습니다.
메뉴 : 모카 시그니처(Mocha Signature)
설명 : 스페셜티 초콜릿과 에스프레소가 만들어내는 풍부하고 고급스러운 조합.
진한 카카오의 깊은 풍미와 커피의 산미가 조화를 이루며 디저트 같은 달콤함을 선사합니다.
메뉴 : 말차 크림 라떼(Matcha Cream Latte)
설명 : 선별된 고급 말차의 풍부한 향과 부드러운 크림층을 더해 완성한 프리미엄 음료.
말차 특유의 쌉쌀한 감칠맛과 고소함이 한 잔에 담겨, 커피를 즐기지 않는 분에게도 사랑받는 특별한 선택입니다.
메뉴 : 허니 브라운 라떼(Honey Brown Latte)
설명 : 자연에서 온 천연 벌꿀을 사용해 은은한 달콤함과 고급스러운 향을 더한 시즌 스페셜.
따뜻함과 깊이가 어우러져 편안하지만 품격 있는 한 잔을 완성했습니다.
메뉴 : 오트 크림 콜드 브루(Oat Cream Cold Brew)
설명 : 깔끔한 콜드 브루 위에 고급 오트 크림을 부드럽게 얹어 풍미의 층위를 더했습니다.
진한 커피의 깊이, 오트의 고소함, 크림의 부드러움이 완벽히 균형을 이루는 프리미엄 비건 라인 메뉴입니다.
